# LaunchList – Serverless Waitlist SaaS (AWS CDK)

Production-grade serverless waitlist app built with AWS CDK (Python).

## Tech Stack
- React frontend (S3 + CloudFront)
- DynamoDB single-table design
- Lambda + API Gateway backend
- Cognito user auth
- SES transactional emails
